# Promises

## 1. Promise based

### 1a

```javascript
Promise.resolve("Resolve1")
  .then(d => {
    console.log(`1-${d}`); // 1-Resolve1
    return d;
  })
  .then(d => {
    console.log(`2-${d}`); // 2-Resolve1
    return Promise.reject("reject1");
  })
  .then(d => {
    console.log(`3-${d}`);
    return Promise.resolve("resolve2");
  })
  .catch(d => {
    console.log(`4-${d}`);
    return `cached simple return`;
  })
  .then(d => {
    console.log(`5-${d}`);
    return d;
  })
  .catch(d => {
    console.log(`6-${d}`);
    return Promise.resolve("resolve3");
  })
  .then(d => {
    console.log(`7-${d}`);
    return "final then";
  })
  .finally(d => {
    console.log("finally", d);
    return 55;
  })
  .then(d => {
    console.log("yet another then", d);
  });
```
